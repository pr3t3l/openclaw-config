CASE FILE — FINANCE TRACKER

Robotin Finance

PERSONAL EXPENSE TRACKING · AIRBNB TAX DEDUCTIONS · TELEGRAM
A personal finance tracker that lives inside Telegram. Send a photo of a receipt or type "$15.50 Uber" — AI parses it, categorizes it, logs it to Google Sheets, and monitors your budget. The differentiator: it automatically flags expenses that might be tax-deductible for rental property owners. Built as a skill inside the CEO agent (Robotin), with a commercialization path toward a standalone SaaS product.
01How It Works: From Receipt to Insight
📸 Send a receipt or text
Take a photo of a receipt or type "$15.50 Uber" in Telegram. The Rule Engine checks first — if it recognizes the merchant (Walmart, Uber, Netflix), it auto-categorizes at $0 cost without calling AI. If the merchant is unknown, the Expense Parser uses AI to extract merchant, amount, date, category, and individual items.
✅ Confirm and log
Robotin shows the parsed result: "Walmart $87.43 Groceries — ¿Confirmar?" You tap yes or correct it. The Transaction Logger writes the confirmed transaction to Google Sheets automatically.
📊 Daily cashflow
Every morning at 7:30 AM (weekdays), you get: "Safe to spend today: $42." The Daily Cashflow module calculates this as: remaining budget minus upcoming bills, divided by days until next payday.
⚠️ Budget alerts
When spending hits 80% of a category budget: "Shopping: $120 of $150 used (80%). 11 days left in month." Plus payment reminders 3 days before due dates for bills and loans.
COST: Rule Engine matches = $0 per parse. AI parsing (unknown merchants) = ~$0.001-$0.003 for text, ~$0.005-$0.01 for photos. Monthly AI analyst report = ~$0.01. Everything else is deterministic and free.
02The 8 Modules

The tracker is built as 8 modules, each with a clear responsibility. The Rule Engine runs first (cheapest path), then AI parsing if needed, then logging, then monitoring. Monthly analysis and bank reconciliation run on their own schedules.

01
Rule Engine
Auto-categorize known merchants via pattern matching. If "Walmart" → Groceries. If "Netflix" → Subscriptions_Other. No AI needed, $0 per parse. Rules stored in rules.json.
02
Expense Parser
AI-powered receipt parsing for unknown merchants. Handles both photos (vision model) and text input. Extracts merchant name, amount, date, category, and individual line items.
03
Transaction Logger
Writes confirmed transactions to Google Sheets via gspread API. Uses batch writes to avoid Google's 429 rate limit errors.
04
Budget Monitor
Checks spending against category budgets. Alerts via Telegram when spending approaches configurable thresholds (default: 80%).
05
Payment Reminder
Checks payment calendar for bills due within 3 days. Sends Telegram alerts with amount, creditor, and account.
06
Daily Cashflow
Calculates "safe to spend today" = (remaining budget - upcoming bills) / days until payday. Runs weekday mornings at 7:30 AM.
07
Monthly Analyst
AI-generated month-end report: spending trends, budget vs actual, category breakdown, and recommendations. The only module besides the parser that uses AI.
08
Reconciliation (v2)
Matches logged receipts against bank CSV exports. Multi-month support, false positive elimination, signed amounts in a dedicated Cashflow_Ledger tab. Supports Wells Fargo checking CSV format.

A 9th module has been designed but not implemented: Batch Receipt Processor. It processes 17+ receipt links at once (Walmart, Target, etc.), replacing generic CSV-imported transactions with detailed line-item breakdowns. Includes deduplication and batch Airbnb confirmation for cleaning/supply items.

03Airbnb Tax Deduction Tracking

This is the feature that differentiates this tracker from every other expense app. For rental property owners, certain expenses — cleaning supplies, linens, repairs, a portion of utilities — are tax-deductible on Schedule E. Most expense trackers don't handle this. Robotin Finance does.

When the parser detects items that could be Airbnb-related (cleaning supplies at Home Depot, property insurance, utility bills), it asks: "Personal or Airbnb?" The answer is tagged in Google Sheets with two fields: tax_deductible (true/false/null) and tax_category (airbnb_supply, airbnb_repair, airbnb_utility, airbnb_insurance, or personal). For utility bills, it asks the split percentage — how much is for the rental vs personal use.

At tax time, the command "Tax summary" generates a total deductible amount by category. "Tax export" produces a CSV formatted for Schedule E that can be sent directly to an accountant.

STATUS: Designed and spec'd in the upgrade document. Not yet implemented. This is the core feature for the SaaS commercialization path.
0418 Categories & 4 Automated Jobs

The tracker uses 18 fixed spending categories, each with a monthly budget and an alert threshold. Categories range from everyday spending (Groceries at $250/mo, Restaurants at $100/mo) to specialized tracking (Subscriptions_AI at $500/mo for OpenAI, Claude, and Gemini subscriptions, or Debt_Interest for credit card interest charges). New categories can be added safely with add_category.sh, which updates the budget config, the parser's category list, and Google Sheets in one atomic operation.

05Google Sheets as Database

All data lives in a Google Sheets workbook called "Robotin Finance 2026" with 8 tabs. Sheets was chosen over a database for three reasons: it's immediately visible (you can open it on your phone and see your spending), it supports manual corrections without tools, and it's free. The trade-off is speed — Google's API has rate limits that required implementing batch writes.

TabPurpose

06Commercialization Path

07Connections to Other Workflows

The Finance Tracker is a skill inside the CEO agent (Robotin) — not a separate bot or service. It's auto-discovered by OpenClaw and runs through the same Telegram bot (@Robotin1620_Bot). It's currently not connected to the PostgreSQL marketing database, Stripe, or the Pipeline's cost tracking. A future integration could pull LiteLLM SpendLogs into the tracker for a unified view of all expenses (subscriptions + API costs + personal spending).
